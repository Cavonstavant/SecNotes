# Webserver enumeration

While searching for attack vectors for this box, I remembered the recent ‘Absolute’ windows box. The foothold in this box was about user enumeration on the AD using metadata in picture files. Navigating to the /about.html page, we can see multiple names that could be users on the AD.

Let’s use username-anarchy to generate some password based on their first and last name

1

❯ /home/kali/username-anarchy/username-anarchy -i usernames_website.txt > password_gen_website.txt

We can then validate if there is a corresponding valid user on the domain using kerbrute
